# 🧠 MAS Fitness & Diet Agentic System

> **AI-powered multi-agent system for personalized fitness and diet planning**

---

## 📌 Overview

The **MAS Fitness & Diet System** is an intelligent **multi-agent AI solution** that generates **personalized diet plans, workout routines, and health reports** based on user data.

It combines:

* 🧠 Multi-Agent Reasoning
* 📊 Nutrition Science
* 🏋️ Fitness Planning
* 📄 Automated PDF Reporting

to deliver a **complete end-to-end fitness solution**.

---

## ✨ Features

* 👤 Smart **User Profiling & Health Analysis**
* 🎯 Automatic **Goal Detection** (Weight Loss, Muscle Gain, Maintenance)
* 🥗 Personalized **Diet Plan Generation**
* 🏋️ Custom **Workout Plan Generation**
* 🧪 Nutritional Analysis (Macros & Micros)
* 🧑‍⚕️ Health Risk Assessment
* 📄 **Dynamic PDF Report Generation**
* 🤖 Multi-Agent Collaboration (Doctor, Dietitian, Nutritionist, Fitness Coach)
* 📚 Knowledge-based recommendations using dietary guidelines

---

## 🏗️ System Architecture

```
User Input
   ↓
User Profiling Agent
   ↓
Goal Detection
   ↓
RAG Knowledge System
   ↓
Multi-Agent Layer
   ├── Doctor Agent
   ├── Dietitian Agent
   ├── Nutritionist Agent
   └── Fitness Agent
   ↓
Decision Fusion
   ↓
Structured Output
   ↓
PDF Report Generator
```

---

## ⚙️ How It Works

### 1. User Profiling

* Collects:

  * Age, Gender
  * Height, Weight
  * Activity Level
  * Food Preference
  * Fitness Level

* Calculates:

  * BMI
  * Calorie Requirements

---

### 2. Goal Identification

* Automatically determines:

  * Fat Loss
  * Muscle Gain
  * Maintenance
  * Health Improvement

---

### 3. Knowledge Integration (RAG)

* Uses:

  * Dietary guidelines
  * Nutritional datasets
* Ensures:

  * Balanced diet
  * Scientific accuracy

---

### 4. Multi-Agent System

#### 🧑‍⚕️ Doctor Agent

* Health analysis
* Risk identification
* Safety constraints

#### 🥗 Dietitian Agent

* Meal planning
* Portion distribution

#### 🧪 Nutritionist Agent

* Macro & micronutrient calculation
* Nutritional optimization

#### 🏋️ Fitness Agent

* Workout plan generation:

  * Weekly split
  * Exercises
  * Sets & reps
  * Progressive overload
  * Recovery strategy

---

### 5. Decision Fusion

* Combines outputs from all agents
* Ensures consistency across:

  * Diet
  * Workout
  * Health constraints

---

### 6. PDF Report Generation

* Fully dynamic (LLM-driven)
* Includes:

  * User profile
  * Fitness goal
  * Diet plan
  * Workout plan
  * Nutritional breakdown
  * Summary & recommendations
